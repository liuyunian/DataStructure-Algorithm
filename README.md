## 测试
### 依赖
1. 测试OS: Ubuntu 18.04 desktop & MacOS 10.15
2. 编译器: gcc 7.4.0
3. 构建工具: cmake 3.10.2

### 编译
1. cd Test  
2. 构建: make
3. 执行: ./build/xxx_test
3. 清除: make clean
